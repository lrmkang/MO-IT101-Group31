**Program Details**
The MotorPH Payroll System is a Java console program that processes employee payroll using data from CSV files. The system first asks the user to log in using a username and password. The two valid users are employee and payroll_staff. If the login credentials are incorrect, the program ends.

If the user logs in as an employee, they can enter their employee number to view their basic details such as employee number, name, and birthday.

If the user logs in as payroll_staff, they can process payroll for one employee or all employees. The system reads attendance records and calculates total hours worked between June and December, based on the work schedule of 8:00 AM to 5:00 PM.

The program then computes the gross salary and applies government deductions including SSS, PhilHealth, Pag-IBIG, and tax. The payroll results are displayed in the console for each cutoff period.

The program uses the csv files for data and calculation.
